ChatGPT注册

2023-04-02

一、关于ChatGPT

ChatGPT（全名：Chat Generative Pre-trained Transformer），是美国OpenAI公司研发的人工智能模型，于2022年11月30日发布。

OpenAI的官方网站是：https://openai.com/ 。
ChatGPT的官方网站是： https://chat.openai.com/ , 也可以用 https://ai.com/ 访问，将自动跳转到前面的地址中。

要使用ChatGPT，需要在OpenAI注册一个账户。这篇博客中，将介绍如何注册OpenAI账户的步骤。

注意：注册ChatGPT需要使用海外服务器及手机号，参考 海外服务器购买指南 和 海外手机号短信验证码接收教程

二、注册账号

1：OpenAI网站

在浏览器中打开 https://openai.com/ ，可以看到OpenAI的官网。

2.进入ChatGPT

在浏览器中打开ChatGPT的入口，地址为 https://chat.openai.com/ ,其页面如下所示

点击“Sign up”按钮，进入注册页面

2.1 使用第三方账号注册

可以直接使用Gmail邮箱和Microsoft账号进行登录，这样就只要跳转第三方验证相应邮箱的密码即可，不用再收取邮箱验证码，更加便捷

2.2 使用邮箱注册

在邮箱地址输入框自己的邮箱，输入后点击“Continue”，继续显示密码输入框。输入自己的邮箱和密码。请确保输入的信息准确无误，这些信息将用于你后续的登录和身份验证。


在你填写完邮箱后之后，你需要验证你的身份。为此，OpenAI会向你提供一个确认邮件。你需要点击邮箱中的“Verify email address”按钮。

3：输入姓和名

验证成功后，OpenAI提供一个页面，要求你填写你的姓和名

输入后，点击“Continue”按钮，就代表你需要同意OpenAI的服务条款和隐私政策，并且确保你是18岁以上。

4：验证手机号

接下来需要验证手机号
在左侧国家列表中选择国家，右侧输入中输入该国家所属的手机号

页面还会询问是否有WhatsApp账号，选择No，然后点击“Send Code via SMS”，即通过短信发送验证码。
收到验证码之后，填入到验证码输入框中，手机就验证成功了。
注：这里可以使用海外亲友的手机号注册，也可以使用第三方平台付费解决，可以参考 SMS-Activate账号注册教程。

5：开始聊天

邮箱和手机号验证都正确的话，就完成注册了，之后就可以登录ChatGPT进入聊天了，地址是 https://chat.openai.com/ 。以下是ChatGPT的聊天界面。

6：订阅ChatGPT Plus（可选）

注册成功后，默认是ChatGPT3.5模型，如果想使用更先进的人工智能，可以充值成ChatGPT Plus，费用20美元/月。

三、总结

注册OpenAI账户本身并不复杂，只需要遵循上述步骤即可。注册成功后，就可以访问OpenAI的ChatGPT。
